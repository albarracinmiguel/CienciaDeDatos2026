This is a Python data science project using a local virtual environment.

- **Python version:** 3.14.2
- **Virtual environment:** `./` (root directory is the venv)

## Activating the environment

```bash

Scripts\Activate.ps1

# On Windows (bash/Git Bash)
source Scripts/activate
```
